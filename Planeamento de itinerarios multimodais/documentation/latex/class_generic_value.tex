\subsection{Detailed Description}
\subsubsection*{template$<$typename Encoding, typename Allocator = Memory\+Pool\+Allocator$<$$>$$>$class Generic\+Value$<$ Encoding, Allocator $>$}

Represents a J\+S\+O\+N value. Use Value for \hyperlink{struct_u_t_f8}{U\+T\+F8} encoding and default allocator. 

A J\+S\+O\+N value can be one of 7 types. This class is a variant type supporting these types.

Use the Value if \hyperlink{struct_u_t_f8}{U\+T\+F8} and default allocator
